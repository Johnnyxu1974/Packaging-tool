Amazon Current SIPP Packaging Requirements (Reference)
Amazon еҪ“еүҚ SIPP еҢ…иЈ…иҰҒжұӮпјҲеҸӮиҖғиҜҙжҳҺпјү

1. Minimum pack size for SIPP eligibility / 1. SIPP еҗҲж јзҡ„жңҖе°ҸеҢ…иЈ…е°әеҜё
  • Minimum pack dimensions: 6.0 Г— 4.0 Г— 0.375 in
  • Packs below this size are not eligible for SIPP certification, because they cannot accommodate the standard 6 Г— 4 in shipping label.
  • жңҖе°ҸеҢ…иЈ…е°әеҜёиҰҒжұӮпјҡ6.0 Г— 4.0 Г— 0.375 иӢұеҜёгҖӮ
  • дҪҺдәҺиҜҘе°әеҜёзҡ„еҢ…иЈ…йҖҡеёёдёҚз¬ҰеҗҲ SIPP и®ӨиҜҒиҰҒжұӮпјҢеӣ дёәж— жі•е®№зәіж ҮеҮҶ 6 Г— 4 иӢұеҜёиҝҗеҚ•ж ҮзӯҫгҖӮ
2. Rigid-pack structural requirements / 2. еҲҡжҖ§еҢ…иЈ…з»“жһ„иҰҒжұӮ
  • Rigid packaging should be six-sided.
  • No protrusions, windows, or cut-outs.
  • Hand holes are allowed only within Amazon-defined limits.
  • Packaging should be properly sealed and designed to avoid unnecessary handling risk.
  • еҲҡжҖ§еҢ…иЈ…еә”дёәе…ӯйқўдҪ“з»“жһ„гҖӮ
  • дёҚеә”жңүзӘҒеҮәзү©гҖҒејҖзӘ—жҲ–жҢ–еӯ”гҖӮ
  • жҸҗжүӢеӯ”д»…еҸҜеңЁ Amazon е…Ғи®ёиҢғеӣҙеҶ…и®ҫзҪ®гҖӮ
  • еҢ…иЈ…еә”е……еҲҶе°ҒеҗҲпјҢе№¶е°ҪйҮҸйҒҝе…ҚйўқеӨ–ж“ҚдҪңе’Ңжҗ¬иҝҗйЈҺйҷ©гҖӮ
3. Flexible packaging / slack-fill requirement / 3. жҹ”жҖ§еҢ…иЈ… / з©әйҡҷйҮҸиҰҒжұӮ
  • Flexible packaging should be right-sized and should not contain excess slack-fill.
  • When lifted at one corner, cumulative slack-fill should not exceed 2 in in any direction.
  • This slack-fill requirement applies mainly to flexible packaging, not rigid packs.
  • жҹ”жҖ§еҢ…иЈ…еә”е°ҪйҮҸиҙҙеҗҲдә§е“Ғе°әеҜёпјҢдёҚеә”еӯҳеңЁиҝҮйҮҸз©әйҡҷгҖӮ
  • еҪ“жӢҺиө·еҢ…иЈ…иўӢдёҖдёӘи§’ж—¶пјҢд»»дёҖж–№еҗ‘зҡ„зҙҜи®Ўз©әйҡҷйҮҸдёҚеә”и¶…иҝҮ 2 иӢұеҜёгҖӮ
  • иҜҘз©әйҡҷйҮҸиҰҒжұӮдё»иҰҒйҖӮз”ЁдәҺжҹ”жҖ§еҢ…иЈ…пјҢе№¶дёҚзӣҙжҺҘзӯүеҗҢдәҺеҲҡжҖ§еҢ…иЈ…зҡ„еӯ”йҡҷзҺҮиҰҒжұӮгҖӮ
4. Note / 4. иҜҙжҳҺ
